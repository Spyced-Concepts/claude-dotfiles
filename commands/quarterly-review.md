# Quarterly Management Review

Run the quarterly formal management review. Work through each section systematically. Flag any item requiring a decision before moving on. Write all decisions and updates to the relevant files before the session closes.

**1. Actions from previous review** — what was committed to? What was done? What carries forward with a new date?

**2. Security posture** — incidents opened this quarter, incidents closed, any recurring patterns? Are all active incidents within SLA?

**3. Risk register** — any new risks this quarter? Review scores on top risks — have likelihoods or impacts changed? Any risks to close or escalate?

**4. SWOT** — any new strengths, weaknesses, opportunities, or threats to record? Any existing items resolved or expired?

**5. Supplier and ethics review** — any supplier that changed materially? Any new dependencies? Any supplier to reconsider on ethics or geo grounds?

**6. Compliance posture** — any new compliance obligations? Any certifications or registrations due? Any policies past review date?

**7. Business performance** — revenue status, pipeline, any material changes to financial position?

**8. Project progress** — status of each active project; top priorities for next quarter.

**9. People and capability** — skills gaps, training, sustainability of current pace?

**10. Next quarter priorities** — define the top 5 priorities with owners and hard dates.

Present findings clearly. Propose file updates; write them once confirmed. Log a completed review entry when done.
