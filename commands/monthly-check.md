# Monthly Lightweight Check

Run the monthly management check. Work through each of these sections, reading relevant files from your knowledge directories. Surface only items that have changed, are flagged, or need a decision — not a full dump of every item.

**1. Security** — any incidents approaching or past SLA? Any new unactioned incidents?

**2. Suppliers / dependencies** — any news flagged this month (CVEs, breaches, ethics issues, financial distress)? Any new dependencies added without review?

**3. Todos and priorities** — any immediate priorities still outstanding from last month? Any scheduled items now overdue? Any carry-forward items sitting for more than 6 weeks — triage: do, defer with new date, or drop?

**4. Financial health** — any unexpected expenses? Any invoices outstanding? Any deferred costs now active?

**5. Project progress** — brief status on each active project: any progress, any blockers?

**6. Compliance pulse** — any new compliance obligations? Any policy review dates overdue?

Present findings concisely. Propose any file updates; write them once confirmed. When done, ask whether to log a completed check entry.

---
*Customise this command for your setup by editing `~/.claude/commands/monthly-check.md`.*
